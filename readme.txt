PLUGIN LOGIN FORM

name: eqeloginfom
version 1.0

Basic login form Sortcode for WordPress, 

INSTALLATION:
create a folder named "eqeloginform", 
insert into eqeloginform.css & eqeloginform.php,
compress the folder .zip.

in wordpress 
upload the eqeloginform.zip as a plugin and activate,
You can now add a shortcode in a block in the editor or like a html tag,
write down [eqeloginform] and its working.

Shortcode:  [eqeloginform]

archive: eqeLoginForm

include: eqeloginform.php y eqeloginform.css as archives and this txt instructions

working in mostly Wordpress themes.
